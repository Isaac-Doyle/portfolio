ATMS

ATMS (Automated Thermal Management System) is a real-time hardware-software control system that stabilizes CPU and GPU temperatures using PID control. It integrates embedded C++ on Arduino for sensor data collection and PWM fan control with Python-based data analysis and visualization, achieving a 22% reduction in thermal variance.

MindTrace is a browser-based mental health check-in app that lets users sign in and log daily mood, energy, stress, and sleep. Users can tag emotions and record notes and gratitude. Entries are saved and visualized through trends like streak tracking and average mood charts, and users can review recent reflections or export/clear their data.
